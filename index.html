El conjunto de datos (Wisconsin) del Cáncer de mama¶

Las caractersticas se calculan a partir de una imagen digitalizada de un aspirado con aguja fina (FNA) de una masa mamaria. Describen las caractersticas de los ncleos celulares presentes en la imagen. El espacio tridimensional es el descrito en: [K. P. Bennett y O. L. Mangasarian: "Discriminacin robusta de la programacin lineal de dos conjuntos linealmente inseparables", Mtodos de optimizacin y software 1, 1992, 23-34].

Del artculo mencionado, leemos que:

"Cada registro representa informacin de seguimiento para un caso cncer de mama. Estos pacientes han sido atentidos por el Dr. Wolberg desde el ao 1984, estn includos slo aquellos casos que exhiben cncer de mama y donde no hay evidencia de metstasis a distancia en el tiempo de diagnstico."

import numpy as np

def divide_en_intervalos(columna,tajada):
    
    minimo = data[columna].min()
    maximo = data[columna].max()
    
    intervalos = np.linspace(minimo, maximo,tajada)
        
    return pd.cut(np.float64(data[columna]), intervalos,precision=4,right=True,include_lowest=True)

Con esta funcion, construimos otra funcion que itera sobre todas las columnas de nuestros datos numericos (quitamos la columna 'Diagnostico'):

In [13]:
def intervalos(tajada):
    
    data_numerical = data.drop(columns=['Diagnostico'])
    intervalos = {}
    
    for column in data_numerical:
        
        intervalos[column] = divide_en_intervalos(column,tajada + 1)      
    
    return intervalos

def histograma_benigno_maligno(intervalo,atributo,xdecimales):
    
    agrupado = data.groupby(intervalo[atributo])[['Benigno','Maligno']].sum();
    ax = agrupado.plot.bar(
                          rot=0, \
                          figsize=(5, 5),\
                          width = 0.9, \
                          stacked=True, 
                          title = atributo,   
                          color=['blue', 'r']
                        );
    
    xlabels = [str(np.around(c.left,decimals=xdecimales)) + ' a ' + str(np.around(c.right,decimals=xdecimales)) \
               for c in  agrupado.index.values];

    ax.set_xticklabels(xlabels,fontsize=8);
    
    return ax

Para medir la simetra, se encuentra el eje mayor, o el acorde ms largo a travs del centro. Luego, medimos la diferencia de longitud entre las lneas perpendiculares al eje mayor del lmite de la celda en ambas direccione.

Otro atributo es la simetra:

La dimensin fractal de una celda se aproxima utilizando la aproximacin de "lnea de costa" descrita por Mandelbrot. El permetro del ncleo se mide utilizando 'reglas' cada vez ms grandes. A medida que aumenta el tamao de la regla, disminuye la precisin de la medicin y disminuye el permetro observado. Al trazar estos valores en una escala logartmica y medir la pendiente descendente se obtiene (el negativo de) una aproximacin a la dimensin fractal. Al igual que con todas las caractersticas de la forma, un valor ms alto corresponde a un contorno menos regular y, por lo tanto, a una mayor probabilidad de malignidad.
